Link to Power of Attorney and Notarial Bond

SOLA allows registered Power of Attorney and Notarial Bonds to be linked into new applications.

Steps

  1. When creating a new application that must reference a Power of Attorney or Notarial Bond, use the Documents search on the Add Document dialog to locate the document to link
  2. Once the desired document is located, use the Select tool to add the document to the application.
  3. Continue with application lodgement as per normal

Notes

  • The Document search will only show registered Power of Attorney and Standard Documents. It is not possible to link to documents that are not registered or have been revoked.
