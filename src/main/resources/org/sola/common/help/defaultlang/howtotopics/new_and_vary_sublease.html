New and Vary Sublease

LAA Lease Services play a role in capturing the details for new subleases and changing subleases.

Cancellation of sublease does not require input from Leases Services and is performed by Registration Services only.

New Sublease

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard, Open the application from your Assigned applications list.
  4. On the Services tab, Start the New Sublease service. The Property Details screen will open
  5. On the Rights / Restrictions tab, choose the Create tool to create a pending sublease right and open the Sublease screen.
  6. Enter all information relevant for the new sublease including (but not limited to), the lease start date, the term, the lessee details and any special conditions. Be aware that the term of the sublease must not exceed the expiry date of parent lease.
  7. To calculate the Ground Rent, select the Land Use, set the Personal Levy and Usable Land % then click the calculate button This will calculate and populate the Ground Rent field. You should check the calculated value and if it is not correct, you should manually update the Ground Rent value.
  8. If there is a subplot defining the area of the sublease, Add the subplot on the Subplot tab. If there is no subplot, leave the Subplot tab empty. Note: To search for an existing subplot, enter the full plot number in the Name First Part Field of the Plot Search screen. E.g. 12301-010
  9. In the Documents section of the Sublease Details screen, Add documents from the application that support the new sublease.
  10. Save & Close the Sublease Details screen.
  11. The Property Details screen will show the new pending Sublease right. Note: If you need to make further changes to the sublease information, select the pending Sublease and click the Edit tool. If you need to remove the pending sublease details completely, use the Remove tool.
  12. Save the Property Details screen and close.
  13. Complete the New Sublease service from the Services tab of Application Details and close the application.
  14. the application so that it can be re-assigned to LAA Registration Services to record the registration details for the transfer

Vary Sublease

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard, Open the application from your Assigned applications list.
  4. On the Services tab, Start the Vary Sublease service. The Property Details screen will open
  5. On the Rights / Restrictions tab, select the Sublease right you intend to change. If the Sublease right is current, the Vary tool will enable. Click Vary to open the Sublease screen.
  6. Update the sublessee details and lease information as required.
  7. In the Documents section of the Sublease Details screen, Add documents from the application that support the changes to the sublease.
  8. Save & Close the Sublease Details screen.
  9. The Property Details screen will show as pending the changes to the Sublease right. Note: If you need to make further changes to the sublease information, select the pending Sublease and click the Edit tool. If you need to remove the pending sublease details completely, use the Remove tool.
  10. Save the Property Details screen and close.
  11. Complete the Vary Sublease service from the Services tab of Application Details and close the application.
  12. the application so that it can be re-assigned to LAA Registration Services to record the registration details for the transfer
