Transfer Lease

Before a lease can be transferred, it is necessary for the Director of Leasing to issue consent for the transfer. Once the consent is issued, the customer(s) involved in the transfer can request the transfer to proceed.

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard, Open the application from your Assigned applications list.
  4. Assess the documentation for the Lease Transfer and confirm the consent for transfer is attached and applies to the property linked to the application.
  5. On the Services tab, Start the Grant of New Lease service
  6. On the Rights / Restrictions tab, choose the Create action. This will open the Lease Details screen.


  7. In the Lessees section of the Lease Details screen, select Add to add a new party record for the lessee. NOTE: If there is more than one lessee, add each individual as a separate party.
  8. On the New party screen, enter the First Name(s), Last name, Gender and Legal status of the new lessee.
  9. If this lessee is to be the nominated Lease account holder, Add the Lease account holder role to the new party record. NOTE: Each lease must have one (and only one) Lease account holder.
  10. Complete other details for the lessee as required.


  11. Save & Close the New party screen.
  12. Add any other lessees as required.
  13. Once all of the new lessees have been added, select the original lessee(s) from the Lessees section of the Lease Details screen and Remove them.


  14. In the Documents section of the Lease Details screen Add documents from the application that support the transfer of the lease.
  15. Save & Close the Lease Details screen.
  16. The Property Details screen will show the new pending Lease right with the new lessee details. Note: If you need to make further changes to the lease information, select the pending lease and click the Edit tool. If you need to remove the pending lease details completely, use the Remove tool.
  17. Save the Property Details screen and close.


  18. Complete the Lease Transfer service from the Services tab of Application Details and close the application.
