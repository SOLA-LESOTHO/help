SOLA Screens

Dashboard and Main Menu

You will be presented with the Dashboard following successful login. The Dashboard lists the applications that are assigned to your user account along with the applications that have yet to be assigned.

Dashboard and Main Menu Details


SOLA Map Viewer

< The SOLA Map Viewer allows you to navigate and interact with the geospatial data managed by SOLA. It can be accessed from the Map > Open Map menu or selecting Map.

SOLA Map Viewer Details


Application Search

< The Application Search screen allows you to find applications of interest for editing or viewing. It can be accessed from the Applications > Search Applications menu or clicking

Application Search


Application Details

< The Application Details screen allows you to enter the details for a new application and/or edit the details of an existing application. To create a new application, click the Applications > New Applications menu or select

Application Details


Application Forms

< The Application Forms screen allows you to quickly access and print the application form templates that must be filled in and submitted by clients when requesting services from LAA. It can be accessed by clicking Application Forms.

Application Forms


Property Search

< The Property Search screen allows you to find and view properties of interest. It can be accessed from the Search > Search Property menu or selecting Property Search

Property Search


Property Details

< The Property Details screen allows you to view or edit property details. You can access the Property Details screen in view only mode by opening a search result from the Property Search screen.
To edit property details, you must first lodge an application that contains a Registration Service such as Grant of New Lease (used to create a new Lease) or Register Mortgage, New Sublease, etc assign the application and then Start the service from the Application Details Service tab. The Land Registration How To section discusses how to process various Registration Services using the Property Details screen

Property Details


Lease Details

< The Lease Details screen contains information about the Lease right for a property. It can be accessed from the Rights and Restrictions tab on the Property Details screen by selecting a Lease right (current, historic or pending) and clicking the toolbar button.

Lease Details


Sublease Details

< The Sublease Details screen is an extension of the Lease Details screen and includes additional tabs for recording the subplot of the sublease and a map tab to show the location of the subplot. It can be accessed from the Rights and Restrictions tab on the Property Details screen by selecting a Sublease right (current, historic or pending) and clicking the toolbar button.

Sublease Details


Mortgage Details

< The Mortgage Details screen contains information about the Mortgage and Sublease Mortgage rights for a property. It can be accessed from the Rights and Restrictions tab on the Property Details screen by selecting a Mortgage or Sublease Mortgage right (current, historic or pending) and clicking the toolbar button.

Mortgage Details


Servitude Details

< The Servitude Details screen contains information about the Servitude rights for a property. It can be accessed from the Rights and Restrictions tab on the Property Details screen by selecting a Servitude right (current, historic or pending) and clicking the toolbar button.

Servitude Details


Document Search

The Document Search screen allows you to find documents of interest for editing or viewing. It can be accessed from the Search > Search Document menu or selecting Document search.

Document Search


Party Search

The Party Search screen allows you to locate parties which include both people (Natural Person) and organizations/groups (Non-natural Person) that are associated to property or property transactions. You can also add, edit, remove or view party details from this screen. It can be accessed by selecting Party or Search > Party.

Party Search


Party Details

The Party Details screen allows you to create new parties as well as edit party details. This screen can be accessed by multiple functions in SOLA including the Party Search screen.

Party Details


User Profile

Every user is able to update their name details and change their password from the User Profile screen. It can be accessed by clicking User profile toolbar icon.

User Profile


Disputes and Court Cases

Disputes between lessees and court cases where the LAA is a party can be recorded on the Disputes and Court Cases screen. This screen can be accessed by clicking Disputes & Court Cases toolbar icon.

Disputes and Court Cases


Finance Export

The Finance Export can be used to create a CSV file containing details about leases required for the LAA Finance System so that the Finance System can produce bills and statements for Ground Rent and manage Ground Rent payments.

Finance Export


Document Registration

Certain document types, specifically Power of Attorney and Notarial Bonds must first be registered in SOLA before they can be referenced by an application to support or justify a property transaction. The Document Registration screen lets you register these official documents as well as deregister them once they are revoked / canceled.

Finance Export


Change to Cadastre

The Change to Cadastre screen allows you to create new plots and subplots for lease purposes. The details for new plots can be obtained from a survey plan either by heads up digitizing from an image of the plan or by entering the coordinates for survey points.

Change to Cadastre


Redefine Cadastre

The Redefine Cadastre screen allows you to change the boundaries of an existing plot both entirely and point by point (adding nodes and changing node coordinates) outside of the receipt of a survey plan. It can be accessed by starting a Redefine Cadastre service from the Application Details Service tab.

Redefine Cadastre
