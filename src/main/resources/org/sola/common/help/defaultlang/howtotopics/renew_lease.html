Renewal of Lease

Renewal of Lease updates the term of the lease

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard, Open the application from your Assigned applications list.
  4. On the Services tab, Start the Renew Lease service
  5. On the General tab, update the Lease Term and Expiration Date.
  6. On the Rights / Restrictions tab, choose the Create action. This will open the Lease Details screen.
  7. In the Documents section of the Lease Details screen, Add documents from the application that support the change in lease term.
  8. Save & Close the Lease Details.
  9. The Property Details screen will show the new pending Lease right. Note: If you need to make further changes to the lease information, select the pending lease and click the Edit tool. If you need to remove the pending lease details completely, use the Remove tool.
  10. Save & Close the Property Details screen.
  11. Complete the Renewal of Lease service from the Services tab of Application Details and close the application.
