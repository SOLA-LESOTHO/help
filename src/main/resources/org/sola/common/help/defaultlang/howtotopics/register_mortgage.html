Register, Cession and Cancel Mortgage

Register Mortgage

When a customer submits a completed application form requesting a registration service from the LAA such as a registration of mortgage, registration of servitude or registration of notarial bond, etc., it will be the responsibility of a Customer Services Representative to lodge an application within SOLA so that the appropriate updates can be made.

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard, Open the application from your Assigned applications list.
  4. On the Services tab, Start the Register Mortgage service. The Property Details screen will open
  5. On the Rights / Restrictions tab, the Mortgage right will be selected as the default Right type. If not, select it from the dropdown list. Click the Create tool to create a pending Mortgage right and open the Mortgage screen.
  6. Fill in the mortgage details and update the Notation text
    1. All fields marked with a red asterisk are mandatory. You may need to estimate some values if the information you have does not contain the specific mortgage details.
    2. Click the ... button on the Lender field to search for the appropriate financial institution. Note that this search will only search parties that have been assigned the Bank role.
    3. The Ranking field indicates the priority of the mortgage
    4. You must include a document reference with the mortgage details. This should be the scanned Deed document.
  7. In the Documents section of the Lease Details screen, Add documents from the application that support the mortgage registration.
  8. Save & Close the Mortgage screen.
  9. The Property Details screen will show the new pending Mortgage right. Note: If you need to make further changes to the mortgage information, select the pending Mortgage and click the Edit tool. If you need to remove the pending mortgage details completely, use the Remove tool
  10. Save changes to Property Details and Complete the service
  11. Use the Validate tool to check the status of the application data. If there are any rule failures, review the application data and correct the issues as appropriate before proceeding to approve the application.
  12. Use the Approve action in the Application action dropdown to approve the application.
  13. Use the Archive action to complete the Application.

Cession (Transfer) of Mortgage

You can use the Cession of Mortgage service to transfer the mortgage from one financial institution (i.e. bank) to another

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard, Open the application from your Assigned applications list.
  4. On the Services tab, Start the Cession of Mortgage service. The Property Details screen will open.
  5. On the Rights / Restrictions tab, select the Mortgage right you intend to change. If the Mortgage right is current, the Vary tool will enable. Click Vary to open the Mortgage screen.
  6. Click the ... button on the Lender field to search for and select the new financial institution. Note that this search will only search parties that have been assigned the Bank role.
  7. In the Documents section of the Mortgage screen, Add documents from the application that support the mortgage cession.
  8. Save & Close the Mortgage screen.
  9. The Property Details screen will show the new pending Mortgage right. Note: If you need to make further changes to the mortgage information, select the pending Mortgage and click the Edit tool. If you need to remove the pending mortgage details completely, use the Remove tool
  10. Save changes to Property Details and Complete the service
  11. Use the Validate tool to check the status of the application data. If there are any rule failures, review the application data and correct the issues as appropriate before proceeding to approve the application.
  12. Use the Approve action in the Application action dropdown to approve the application.
  13. Use the Archive action to complete the Application.

Cancel Mortgage

You can use the Cancel Mortgage service to discharge a mortgage on a property.

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard, Open the application from your Assigned applications list.
  4. On the Services tab, Start the Cancel Mortgage service. The Property Details screen will open.
  5. On the Rights / Restrictions tab, select the Mortgage right you intend to cancel. If the Mortgage is current and there are no pending changes for the Mortgage, the Extinguish tool will enable. Click Extinguish to open the Mortgage screen.
  6. Update the details as appropriate and set the Notation text.
  7. Click Create and Save and close the Mortgage Details screen.
  8. The Property Details screen will show a new pending Mortgage right. Note: If you need to make further changes to the mortgage information, select the pending Mortgage and click the Edit tool. If you need to remove the pending mortgage details completely so that the mortgage is not cancelled, use the Remove tool.
  9. Save changes to Property Details and Complete the service
  10. Use the Validate tool to check the status of the application data. If there are any rule failures, review the application data and correct the issues as appropriate before proceeding to approve the application.
  11. Use the Approve action in the Application action dropdown to approve the application.
  12. Use the Archive action to complete the Application.
