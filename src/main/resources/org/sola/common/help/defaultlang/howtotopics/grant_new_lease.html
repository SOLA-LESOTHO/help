Grant of New Lease

It is the responsibility of LAA Lease Services to assess the information provided for the new lease and either prepare the new lease documentation or reject the new lease request. If the new lease is granted, the Letter of Offer and the Lease will be sent to the lessee(s) for their signature(s). Once the lessees have signed the new lease documentation and paid the necessary fees, LAA Registration Services will formally register the new lease.

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard, the application from your Assigned applications list.
  4. Assess the documentation provided for the Grant of New Lease application and determine if it the grant can continue. If so, complete the steps below. If the grant request must be rejected, skip the remaining steps and refer to the Rejecting a Lease Grant section instead.
  5. On the Services tab, Start the Grant of New Lease service
  6. On the Plot tab, choose Select from application to open the Plot Selection dialog


  7. In the Plot Selection dialog, Select the plot that will be used for the lease. If the plot you require for the lease is not displayed in the Plot Selection dialog, you can use the Search on the Plot tab to locate the appropriate plot instead
  8. On the Plot tab, set the Road Class, Land Grade and Valuation Amount for the lease as required.
  9. On the Rights / Restrictions tab, choose the Create action. This will open the Lease Details screen.
  10. Enter all information relevant for the new lease including (but not limited to), the lease start date, the term, the lessee details and any special conditions. You should also link any documentation from the application that is relevant to the granting of the lease.
  11. To calculate the Ground Rent, select the Land Use, set the Personal Levy and Usable Land % then click the calculate button This will calculate and populate the Ground Rent field. You should check the calculated value and if it is not correct, you should manually update the Ground Rent value.
  12. Once you have completed entering all of the necessary information for the new lease, you can check the Letter of Offer and Lease documents using the Print drop down. This will generate the documents for you to review. You should print these documents as must be signed by the Director of Leasing and sent to the lessee(s).


  13. If the details are in lease details and documentation is in order, Save & Close the Lease Details screen.
  14. The Property Details screen will show the new pending Lease right with the new lessee details. Save the Property Details screen and close.
  15. From the Dashboard, select the application in the Assigned applications list and click the Unassign tool. This will remove you as the assigned user and display the application in the Unassigned applications list.
  16. Notify the Lease Services Manager that the application is available for review by placing the Letter of Offer and Lease notices in their inbox.

    17. Lease Services Manager Review

    Before the Letter of Offer and Lease can be sent to the lessee(s), the new lease details must be checked by the Lease Services Manager. The Lease Services Manager will:

    1. Using the Dashboard, Assign the application to themselves and Open it.
    2. Assess all documentation provided for the Grant of New Lease application and determine if all details are in order. If so..
    3. From the Services tab, Start the Grant of New Lease service
    4. From the Rights / Restrictions tab, select the new pending lease right and click the action.
    5. Check the details recorded for the lease match the documentation provided. If not, the Lease Services Manager can either reassign the application back to the staff member that recorded the lease details or correct the lease details themselves. If the Lease Services Manager makes changes to the lease details they should reprint the Letter of Offer and Lease notices.
    6. If everything is in order, Save & Close the Lease Details screen.
    7. The Property Details screen will show the new pending Lease right with the new lessee details. Note: If you need to make further changes to the lease information, select the pending lease and click the Edit tool. If you need to remove the pending lease details completely, use the tool.
    8. Save the Property Details screen and close.
    9. Complete the Grant of New Lease service from the Services tab of Application Details and close the application.
    10. Unassign the application so that it can be re-assigned to LAA Registration Services once the lessee(s) have signed the lease documents and paid the necessary fees.
    11. 11. Place the Letter of Offer and Lease notices in the Director of Leasing’s inbox for them to sign and forward to the lessee(s).

    Rejecting a Lease Grant

    In some situations it may be necessary to reject the grant for a new lease (e.g. lessee under 18, invalid certificates, etc.). Where you have determined the lease must be rejected

    1. On the Services tab, Start the Grant of New Lease service
    2. On the Plot tab, choose Select from application to open the Plot Selection dialog
    3. In the Plot Selection dialog, Select the plot that will be used for the lease. If the plot you require for the lease is not displayed in the Plot Selection dialog, you can use the Search on the Plot tab to locate the appropriate plot instead.
    4. On the Rights / Restrictions tab, choose the Create action. This will open the Lease Details screen.
    5. Use the Print dropdown and select the Rejection Letter menu option
    6. Enter the reason for rejecting the application in the Rejection Reason dialog and click Save



    7. The Rejection Letter will be generated. Save a PDF copy of the letter and print a hard copy of letter.
    8. Close the Lease Details screen without saving.
    9. Close the Property Details screen without saving.
    10. On the Application Details screen, go to the Document tab and Add a new document attaching the PDF copy of the Rejection Letter.
    11. Save the Application Details screen and close.
    12. From the Dashboard, select the application in the Assigned applications list and click the Unassign tool. This will remove you as the assigned user and display the application in the Unassigned applications list.
    13. Notify the Lease Services Manager that the application should be

    The Lease Services Manager will:

    1. Using the Dashboard, Assign the application to themselves and Open it.
    2. Assess all documentation provided for the Grant of New Lease application and determine if all the rejection is justified. If so…
    3. From the Application Details screen, use the Application action drop down to Cancel the application. Cancellation of the application is required in case there is a subsequent request to grant a lease over the same plot.
    4. Unassign the application so that it can be re-assigned to LAA Registration Services once the lessee(s) have signed the lease documents and paid the necessary fees.

    If the Lease Services Manager determines that rejection of the application is not justified, they can Remove the Rejection Letter from the Documents tab of the Application Details screen and reassign the application back to the Lease Services staff member to continue capture of the lease details.
