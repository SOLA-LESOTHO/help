Property search

The Property Search screen allows you to find and view properties of interest. It can be accessed from the Search > Search Property menu or selecting Property Search



You can search by any combination of Name first part (grid number), Name last part (plot number) Lease Number or Lessee (name). These fields also support partial and full matching.
To view the details for a property, select the property in the Search results list and click the button.
