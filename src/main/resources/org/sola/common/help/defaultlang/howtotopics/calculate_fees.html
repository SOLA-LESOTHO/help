Calculate Fees

TO BE REVISED

To calculate the fee for an application go to the Fees tab in Application Details and click the Calculate Fee button



Using Calculate Fee will not save any changes to the application. This allows you to determine the fees that will apply to a new application prior to lodging it. Fee calculation is also performed automatically when the application is lodged.
