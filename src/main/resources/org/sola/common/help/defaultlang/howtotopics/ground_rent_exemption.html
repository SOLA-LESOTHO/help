Ground Rent Exemption Application

A Ground Rent Exemption Certificate must be generated and registered on the lease.

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard, the application from your Assigned applications list.
  4. On the Services tab, Start the Ground Rent Exemption service
  5. On the Rights / Restrictions tab, the select the current Lease right and click the Vary tool. This will open the Lease Details screen.
  6. Edit the Ground Rent and set it to 0
  7. 7. Use the Print dropdown to generate the Ground Rent Exemption Certificate. Save the Certificate as a PDF.
  8. In the Documents section of the Lease Details screen, Add documents from the application that support the exemption. You should also attach the Ground Rent Exemption Certificate generated above.
  9. Save & Close the Lease Details screen.
  10. The Property Details screen will show the pending changes to the lease right with the new ground rent details. Note: If you need to make further changes to the sublease information, select the lease with pending changes and click the Edit tool. If you need to remove the pending ground rent details completely, use the Remove tool.
  11. Save the Property Details screen and close.
  12. Complete the Grant of New Lease service from the Services tab of Application Details and close the application.
  13. Unassign the application so that it can be re-assigned to LAA Registration Services once the lessee(s) have signed the lease documents and paid the necessary fees.
