Register, Vary and Cancel Servitudes

Servitude (a.k.a. easement) is a right to use the property of another without possessing it. A good example is a right of way where access to a property may be granted across the property of another.

Servitudes over property can be in favor of one or more named right holders, but are more commonly in favor of another property (e.g. Right of Way). SOLA provides the Servitude right to support the recording servitude over the property subject to the servitude.

Register Servitude

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the Assign tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard Open the application from your Assigned applications list.
  4. Assess all documentation provided for the Grant of New Lease application and determine if all details are in order. If so..
  5. From the Services tab, Start the Register Servitude service. The Property Details screen will open.
  6. On the Rights / Restrictions tab, the Servitude right will be selected as the default Right type. If not, select it from the dropdown list. Click Create tool to create a pending Servitude right and open the Servitude screen.
  7. In the Servitude screen fill in the Registration date, add a document that describes the servitude and update the Notation text to indicate what the servitude is and the property, properties or right holder(s) it is in favour of.
  8. Enter any lessees affected by the servitude in the Lessees section.
  9. Create & Save and close the Servitude screen.
  10. The Property Details screen will show the new pending Lease right with the new lessee details. Note: If you need to make further changes to the lease information, select the pending servitude and click the Edit tool. If you need to remove the pending servitude details completely, use the tool.
  11. Save the Property Details screen and Complete the service
  12. Use the Validate tool to check the status of the application data. If there are any rule failures, review the application data and correct the issues as appropriate before proceeding to approve the application.
  13. Use the Approve action in the Application action dropdown to approve the application.
  14. Use the Archive action to complete the Application.

Cancel Servitude

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the Assign tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard Open the application from your Assigned applications list.
  4. From the Services tab, Start the Cancel Servitude service. The Property Details screen will open.
  5. On the Rights / Restrictions tab, select the Servitude right you intend to cancel. If the Servitude is current and there are no pending changes for the Servitude, the Extinguish tool will enable. Click Extinguish to open the Servitude screen.
  6. In the Documents section, Add documents from the application that support the change to the servitude.
  7. Click Create & Save and close the Servitude screen.
  8. Use the Validate tool to check the status of the application data. If there are any rule failures, review the application data and correct the issues as appropriate before proceeding to approve the application.
  9. Use the Approve action in the Application action dropdown to approve the application.
  10. Use the Archive action to complete the Application.
