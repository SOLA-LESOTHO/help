Add Parent Property

You can use the Add Parent Property feature to create a relationship between two properties. The main purpose of this feature is to setup prior title references. In Lesotho, when a lease is cancelled (i.e. terminated or surrendered), it is possible for a new lease to be created in the same area of land as the cancelled lease. The Add Parent Property feature can be used in this case to link the cancelled lease to the new lease, thereby providing a history of leases in that area.

The feature is available when processing a Grant of New Lease service by clicking the Add tool on the Property history tab of the Property Details screen.

Steps

  1. While performing a Grant of New Lease service, go to the Property history tab and click the Add tool.
  2. Search for the parent property and Select it. The property you select must have a historic status (i.e. it must have been previously cancelled).


  3. The Property history tab will be displayed showing the parent property you selected.


  4. You can Open the parent property to view its details or Remove it if necessary
  5. Save the Property Details screen and continue recording the lease details for the new lease as required.
