Cancel Sublease

To surrender or terminate a sublease lease, use the Cancel Sublease service.

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard, Open the application from your Assigned applications list.
  4. On the Services tab, Start the Cancel Sublease service. The Property Details screen will open
  5. On the Rights / Restrictions tab, select the Sublease right you intend to cancel. If the Sublease right is current and there are no pending changes for the Sublease, the Extinguish tool will enable. Click Extinguish to open the Sublease screen.
  6. In the Documents section of the Sublease Details screen, Add documents from the application that support the cancellation of the sublease.
  7. On the Registration tab, enter the Registration Date, Registration Number and any applicable fee information.
  8. Save & Close the Sublease Details screen.
  9. The Property Details screen will show a new pending change against the Sublease right. Note: If you need to make further changes to the lease information, select the pending lease and click the Edit tool.
  10. Save the Property Details screen and close.
  11. Complete the Cancel Sublease service from the Services tab of Application Details and close the application.
  12. Use the Validate tool to check the status of the application data. If there are any rule failures, review the application data and correct the issues as appropriate before proceeding to approve the application.
  13. Use the Approve action in the Application action dropdown to approve the application.
  14. Use the Archive action to complete the Application.

The sublease right will now display as a Historic right for the property.
