Change Lessee Name

Lessee names may change due to marriage or deed poll. The process for the Change of Lessee Name service is same as that for Transfer Lease. Refer to the steps described in the Transfer Lease section for guidance.

IMPORTANT: You must use the Add to add a new lessee (i.e. party) with the new name and Remove to remove the original lessee details. Do not use Edit to update the original lessee name as this will prevent an accurate history of lessee names from being maintained.
