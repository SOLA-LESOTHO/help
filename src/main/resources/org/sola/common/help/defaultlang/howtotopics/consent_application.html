Consent Application

The Director of Leasing must first issue consent before a lease can be transferred or have a sublease or private servitude registered against it.

LAA Lease Services are responsible for reviewing the details of the consent application and producing the consent notice.

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard, Open the application from your Assigned applications list.
  4. On the Services tab, Start the Consent Application service. The Consent Application screen will open


  5. Complete the details for the Consent including the Transaction Type, Registration Date, Lessees and Recipient details
  6. Once all information relevant for the consent has been entered, use the Print Certificate button to generate and print the consent notice.
  7. Close the Consent Application screen

  8. Open the application again and Complete the Consent Application service from the Services tab of Application Details.
  9. Use the Validate tool to check the status of the application data. If there are any rule failures, review the application data and correct the issues as appropriate before proceeding to approve the application.
  10. Use the Approve action in the Application action dropdown to approve the application.
  11. Use the Archive action to complete the Application

If the consent is not approved, open the application and Cancel it
