Registration on Lease

Nearly all of the lease services require LAA Registration Services to formally register the changes made to the lease. Registration occurs after LAA Lease Services has completed the preparation of the lease documents and the customer has signed and returned the relevant paperwork as well as paid the required fees.

Steps

  1. From the Dashboard, select the application in the Unassigned applications list and click the tool. This will open the Application Assignment dialog.
  2. Click the Assign button to assign the application. When the Assign action is complete, you will be returned to the Dashboard and the assigned application will appear in your Assigned applications list.
  3. From the Dashboard, Open the application from your Assigned applications list.
  4. If documents have been signed and returned by the Lessee (e.g. Lease Deed, Surrender of Lease, etc.) then go to the Documents tab and create and attached scanned copies of the documents to the application.
  5. On the Services tab, Start the Registration on Lease service. The Property Details screen will open
  6. On the Rights / Restrictions tab, the select the pending Lease (or pending Sublease) right and click the Edit tool. This will open the Lease Details screen.
  7. Go to the Registration tab and enter the Registration Date, Registration Number and any fee details.
  8. In the Documents section of the Lease Details screen, Add any new documents from the application that are relevant for the lease service (e.g. the signed Lease Deed for Grant of New Lease service)
  9. Save & Close the Lease Details screen.
  10. The Property Details screen will show the pending Lease right with the new lessee details. Note: If you need to make further changes to the lease information, select the pending lease and click the Edit tool.
  11. Save the Property Details screen and close.
  12. Complete the Registration on Lease service from the Services tab of Application Details and close the application.
  13. Use the Validate tool to check the status of the application data. If there are any rule failures, review the application data and correct the issues as appropriate before proceeding to approve the application.
  14. Use the Approve action in the Application action dropdown to approve the application.
  15. Use the Archive action to complete the Application.
