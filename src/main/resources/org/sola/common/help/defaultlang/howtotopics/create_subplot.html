Create Subplot

You can create a subplot to define the area of a sublease in much the same way as creating a plot. The main difference is to select Subplot instead of Parcel as the Type on the Plot properties dialog.

Steps

  1. Start the Change to Cadastre Service to access the Change to Cadastre screen.
  2. Select any target plots and create any new survey points or plots as required.
  3. Create the new subplot using the Create New Plot tool On the Plot properties dialog, choose Subplot as the Type instead of Parcel. The new subplot will automatically get assigned the appropriate subplot number when the Change to Cadastre is saved.
  4. Continue to capture any other details for the survey and Complete the Change to Cadastre service as per usual.
