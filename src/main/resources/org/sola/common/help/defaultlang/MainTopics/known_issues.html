Known issues

The known issues for the SOLA Lesotho version are:

Unable to Login

Cause

Description

Resolution

Invalid user credentials

Enter valid user credentials. If you cannot remember your user credentials, contact IT Support to reset your password

User account is disabled

Contact IT Support to verify if your account is enabled.

Application services are unavailable

The SOLA Web Start applications use web services on the SOLA Application Server to process information. If the web services are unavailable due to a problem with the network or the SOLA Application Server, you will receive the user name and password could not be verified message.

Contact IT Support to restart the SOLA Application Server.

SOLA Desktop shortcut does not appear

If you have installed Java 7 update 7 (build 1.7.0_07) you may find that the SOLA Desktop shortcut does not appear on your desktop. This is an issue with update 7. Upgrade to a later version of Java 7 or downgrade to an earlier version (e.g. Java 7 update 3).
